# Orbis — Premium Hackathon Platform

Orbis는 개발자들이 해커톤을 탐색하고, 팀을 구성하며, 성과를 기록할 수 있는 프리미엄 플랫폼입니다.

## 주요 기능
- **해커톤 탐색**: 상세 필터링과 검색 기능
- **팀 빌딩 (Camp)**: 기술 스택 기반 팀원 모집 및 지원
- **명예의 전당**: 활동 포인트를 기반으로 한 랭킹 시스템
- **통합 스토리지**: 관심 해커톤 북마크 및 지원 현황 관리 (Local Storage 기반)
